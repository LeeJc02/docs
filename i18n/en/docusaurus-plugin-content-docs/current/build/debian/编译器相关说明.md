# Compiler-related Information

RevyOS comes pre-packaged with an optimized compiler that primarily supports `rv64gc`.

## General Extensions

| Supported Optimizations | gcc-14 | clang-19 | clang-20 |
| :--: | :--: | :--: | :--: |
| Zfh  | ✅ | ✅ | ✅ |
| V    | ✅ | ✅ | ✅ |
